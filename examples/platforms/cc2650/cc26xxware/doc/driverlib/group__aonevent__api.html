Detailed Description

Introduction

The event fabric consists of two event modules. One in the MCU power domain (MCU event fabric) and the other in the AON power domain (AON event fabric). The MCU event fabric is one of the subscribers to the AON event fabric. For more information on MCU event fabric, see MCU event API.

The AON event fabric is a configurable combinatorial router between AON event sources and event subscribers in both AON and MCU domains. The AON event fabric has three main event subscribers: AON WUC, AON RTC, and MCU event fabric. The API to control the AON event fabric configuration can be grouped based on the event subscriber to configure:

  • AON WUC receives two programmable event lines from AON event fabric. One for MCU wake-up events and one for AUX wake-up events. For more information, see AON WUC API.
    • MCU wake-up event
      • AONEventMcuWakeUpSet()
      • AONEventMcuWakeUpGet()
    • AUX wake-up event
      • AONEventAuxWakeUpSet()
      • AONEventAuxWakeUpGet()
  • AON RTC receives a single programmable event line from the AON event fabric. For more information, see AON RTC API.
    • AONEventRtcSet()
    • AONEventRtcGet()
  • MCU event fabric receives a number of programmable event lines from the AON event fabric. For more information, see MCU event API.
    • AONEventMcuSet()
    • AONEventMcuGet()

uint32_t AONEventAuxWakeUpGet ( uint32_t  ui32AUXWUEvent)

Get event source for the specified AUX wakeup programmable event.

The AON event fabric has a total of three programmable events that can wake-up the AUX domain. The events are forwarded to the wakeup controller (WUC).

Parameters
ui32AUXWUEventis one of three programmable AUX wakeup event sources.
  • AON_EVENT_AUX_WU0
  • AON_EVENT_AUX_WU1
  • AON_EVENT_AUX_WU2
Returns
Returns an event source for the event AON fabric, i.e. one of the following:
  • AON_EVENT_DIO0 : Edge detect on DIO0. See hw_device.h
  • ...
  • AON_EVENT_DIO31 : Edge detect on DIO31. See hw_device.h
  • AON_EVENT_IO : Edge detect on any DIO
  • AON_EVENT_RTC_CH0 : RTC channel 0
  • AON_EVENT_RTC_CH1 : RTC channel 1
  • AON_EVENT_RTC_CH2 : RTC channel 2
  • AON_EVENT_RTC_CH0_DLY : RTC channel 0 - delayed event
  • AON_EVENT_RTC_CH1_DLY : RTC channel 1 - delayed event
  • AON_EVENT_RTC_CH2_DLY : RTC channel 2 - delayed event
  • AON_EVENT_RTC_COMB_DLY : RTC combined delayed event
  • AON_EVENT_RTC_UPD : RTC Update Tick (16 kHz signal, i.e. event line toggles value every 32 kHz clock period)
  • AON_EVENT_JTAG : JTAG generated event
  • AON_EVENT_AUX_SWEV0 : AUX Software triggered event #0
  • AON_EVENT_AUX_SWEV1 : AUX Software triggered event #1
  • AON_EVENT_AUX_SWEV2 : AUX Software triggered event #2
  • AON_EVENT_AUX_COMPA : Comparator A triggered (synchronized in AUX)
  • AON_EVENT_AUX_COMPB : Comparator B triggered (synchronized in AUX)
  • AON_EVENT_AUX_ADC_DONE : ADC conversion completed
  • AON_EVENT_AUX_TDC_DONE : TDC completed or timed out
  • AON_EVENT_AUX_TIMER0_EV : Timer 0 event
  • AON_EVENT_AUX_TIMER1_EV : Timer 1 event
  • AON_EVENT_BATMON_TEMP : BATMON temperature update event
  • AON_EVENT_BATMON_VOLT : BATMON voltage update event
  • AON_EVENT_AUX_COMPB_ASYNC : Comparator B triggered. Asynchronous signal directly from the AUX Comparator B
  • AON_EVENT_AUX_COMPB_ASYNC_N : Comparator B not triggered. Asynchronous signal directly from the AUX Comparator B
  • AON_EVENT_NONE : No event, always low
